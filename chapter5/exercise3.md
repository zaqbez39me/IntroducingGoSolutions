# Exercise 3

## Statement

Given the following array, what would `x[2:5]` give you?

```go
x := [6]string{"a", "b", "c", "d", "e","f"}
```

## Solution

The `x[2:5]` will create a slice of this array from 3rd element to 6th. The length of slice will be 3. The content will
be equal to `["c", "d", "e"]`

## Answer

`["c", "d", "e"]`
